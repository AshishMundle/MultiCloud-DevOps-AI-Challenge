## 💡 Key Learnings from Session 40

> **1.** Ansible is agentless — uses only SSH with no software required on target servers
> **2.** Terraform provisions infrastructure Ansible configures what was just provisioned
> **3.** YAML indentation is critical — use exactly 2 spaces and never use tabs in YAML
> **4.** Ansible is idempotent — running the same playbook twice produces the same result
> **5.** 90 percent of Fortune 500 companies use Ansible for configuration management daily
