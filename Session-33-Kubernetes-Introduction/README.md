## 💡 Key Learnings from Session 33

> **1.** Kubernetes is container orchestration — it manages containers at scale automatically
> **2.** Control Plane has API Server etcd Scheduler and Controller Manager components
> **3.** Worker Node runs Pods with kubelet kube-proxy and container runtime components
> **4.** kubectl get pods -n namespace is your most used Kubernetes daily command
> **5.** Kubernetes self-heals — if a pod crashes it automatically restarts it within seconds
