## 💡 Key Learnings from Session 22

> **1.** Git tracks every change ever made — you can recover any version from full history
> **2.** Fork creates your own copy of someone else's repo Clone downloads it to local machine
> **3.** git status is the most used command — run it before every single commit
> **4.** Version control is required for every project even solo ones — no exceptions
> **5.** GitHub Actions is free CI/CD directly in GitHub — learn alongside Jenkins
