## 💡 Key Learnings from Session 42

> **1.** Roles are reusable Ansible code — tasks vars templates handlers in standard structure
> **2.** ansible-galaxy install geerlingguy.nginx installs a production-grade nginx role instantly
> **3.** Jinja2 templates use double curly brace variable in config files for dynamic configuration
> **4.** Handlers run only when notified — ideal for restarting services after config file changes
> **5.** ansible-vault encrypts secrets in playbooks — never store passwords in plaintext YAML
