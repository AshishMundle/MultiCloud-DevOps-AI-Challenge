> **2.** Shared Libraries allow reusing pipeline code across multiple Jenkinsfiles in org
> **3.** Blue Ocean gives beautiful visual representation of all pipeline stages and logs
> **4.** environment block in Jenkinsfile sets variables available to all stages globally
> **5.** withCredentials block safely uses secrets without ever exposing them in build logs
