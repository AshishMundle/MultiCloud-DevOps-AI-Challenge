> **2.** Ingress controller routes external HTTP traffic to internal services by path or hostname
> **3.** PersistentVolumeClaim is the Kubernetes way of requesting disk storage for stateful apps
> **4.** Namespace separates environments cleanly: kubectl apply -f . -n staging for staging
