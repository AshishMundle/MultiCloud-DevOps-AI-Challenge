## 💡 Key Learnings from Session 08

> **1.** chmod 600 ~/.ssh/id_rsa is MANDATORY — SSH client refuses connection otherwise
> **2.** 755 means rwxr-xr-x: owner full access others read and execute — standard for scripts
> **3.** SSH key auth is more secure than passwords and required for all cloud virtual machines
> **4.** apt update downloads package list apt install actually installs the software
> **5.** 403 Forbidden is permissions issue 404 is file missing 500 is server crash
