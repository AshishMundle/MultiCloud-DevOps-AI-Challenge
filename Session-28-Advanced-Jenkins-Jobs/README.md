## 💡 Key Learnings from Session 28

> **1.** Multi-branch pipeline automatically creates jobs for every branch in the repository
> **2.** Credentials plugin stores secrets encrypted — never hard-code passwords in Jenkinsfiles
> **3.** Jenkins backup means backing up JENKINS_HOME directory — schedule daily cron job
> **4.** Job DSL creates Jenkins jobs as code — version control your entire Jenkins configuration
> **5.** Folder plugin organizes jobs by project and environment — essential for large teams
