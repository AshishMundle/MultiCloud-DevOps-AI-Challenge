## 💡 Key Learnings from Session 07

> **1.** df -h is the first command to run on any server — full disk means server crash
> **2.** systemctl enable nginx starts on every boot systemctl start nginx starts right now
> **3.** top shows real-time CPU and memory — press P to sort by CPU usage
> **4.** tail -f /var/log/syslog is your live production monitoring window in terminal
> **5.** grep -r error /var/log finds problems across all log files instantly
