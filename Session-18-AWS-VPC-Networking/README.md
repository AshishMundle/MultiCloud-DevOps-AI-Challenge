## 💡 Key Learnings from Session 18

> **1.** VPC is your private data center inside AWS — completely isolated network environment
> **2.** CIDR /16 is 65536 IPs /24 is 256 IPs /32 is 1 IP — know these for every interview
> **3.** NACL is stateless so you must allow both inbound and outbound Security Groups are stateful
> **4.** Public subnet has route to Internet Gateway Private subnet routes through NAT Gateway
> **5.** VPC Peering connects two VPCs privately without any internet traffic between them
